#+TITLE: My dwm Configuration

* Overview
This repository contains my personal configuration of [[https://dwm.suckless.org/][dwm]] (Dynamic Window Manager).  
It's a minimal, keyboard-driven tiling window manager written in C and maintained by the [[https://suckless.org][suckless.org]] community.

My setup aims to keep dwm minimal while adding some essential usability patches and aesthetic improvements.

* Features
- Patched and configured dwm (Dynamic Window Manager)
- Keybindings optimized for productivity

* Applied Patches
List of patches currently applied:
- [[https://dwm.suckless.org/patches/anybar/][anybar]] to apply other statusbar
- [[https://github.com/rose-pine/dwm][rose-pine]] colorscheme for dwm
- [[https://dwm.suckless.org/patches/fibonacci/][fibonacci]] for fibonacci type layout(Mod+r)
- [[https://dwm.suckless.org/patches/ewmhtags/][wemhtags]] to support xworksapaces

* Usage
- =Mod + Enter=: Launch ghostty
- =Mod + d=: Launch rofi
- =Mod + b=: Toggle bar visibility
- =Mod + Shift + q=: Quit dwm
- =Mod + Shift + Space=: Toggle floating mode
- =Mod + [1-9]=: Switch to workspace
- =Mod + r=: Switch to fibonacci layout
- =Mod + Shift + [1-9]=: Move window to workspace
